# Budgets Control

An application to have a control of budgets, expenses and advances in an enterprise.

	- Vue.js
	- Vuex
	- Vue Router
	- Buefy
	- Laravel
	- Laravel Passport

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
